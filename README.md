# EcoMap GIS - Lahore

A web-based GIS application to help users locate garbage containers and dumping yards in Lahore, Pakistan.

## Features (v1.0)
- Interactive map of Lahore
- Locations of garbage containers and dumping yards
- Filter by type (Container vs Dumping Yard)
- Mobile-responsive design

## Setup
1. Clone the repository
2. Open `index.html` in your browser

## User Roles
- **Guest**: View and search
- **Agent**: Add/Edit markers (Login: `agent`/`123`)
- **Admin**: Full access (Login: `admin`/`123`)
